# TODO Backlog

## Defines
- Unscheduled task lists with issue-promotion history.
- 明確な計画を立てる前に発生した「やるべきこと」の一次置き場と、Issue起票済みの履歴を管理する。
- 未起票の項目は流動的に追加・統合・分割・撤回できる運用対象。

## Promotion rule
- 「今やる」と決めた項目のみ Issue 化する。
- Issue 化した項目は `backlog/INDEX.md` と `backlog/issues/` に反映する。
- Issue 化後は `未起票` から `ISSUE起票済み` へ移動する（削除しない）。
